# 파일 업로드2 예제
**Spring Boot**, **TypeScript** 를 활용한 파일 업로드 예제

---
## 사용 기술
- FrontEnd
  - **Next.js**
  - **TypeScript**
  - **Axios**
- BackEnd
  - **JDK 17.0.12 LTS**
  - **Spring Boot 3.4.1**
  - **Spring Data JPA**
  - **H2 Database**
  - **Mockito, JUnit5**
